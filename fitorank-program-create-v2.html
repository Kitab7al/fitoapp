Система весов
Подход А: автор задаёт % от 1RM или RPE. Атлет вводит свой 1RM один раз — дневник рассчитывает рабочие веса автоматически. Форматы можно комбинировать.
Основной формат записи весов в программе
📊

🎯

Вспомогательные упр.
💡 Можно комбинировать форматы — для каждого упражнения в структуре выбирается свой тип. Например: жим лёжа в % от 1RM, изоляция в RPE.
Определение 1RM покупателя
Как атлет узнает свой 1RM перед началом программы?
При выборе «Тестовая неделя» — дневник начнётся с протокола определения 1RM, результат автоматически подставится во все расчёты.

Рекомендация для натуралов: Основные подъёмы 75–85%, вспомогательные RPE 7–8. Не используй 90%+ чаще 1 раза в 2 недели — риск травмы и перетренированности.
